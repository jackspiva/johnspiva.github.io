Projects

Worksheet Interlace - 2017

worksheet-project.herokuapp.com

This is a the web application portion of my senior capstone project that I am doing with two classmates. The web application displays and visualizes data gathered from classroom worksheets completed by students using image processing.

Julia Sets - 2016

This is a program I built with a partner for a class called “Numerical Analysis.” The program is written in MATLAB and can plot and visualize Julia sets. It can animate the formation of a Julia set as well as the development of a changing one.

Trip Explorer - 2016

I built this application with three peers using React Native and D3.js at a hackathon. The application uses the Trip Advisor API to suggest destinations based on what a user wants to do. The suggestions are displayed u sing a force-directed graph built with D3.js.

Bostunes - 2015

This is a web application I built with three other students using Node.js, Express, and MongoDB. The application provides users with concert recommendations based on the songs on their “Starred” playlist on Spotify. The application allows users to add concerts to their Google calendar and get directions to concert venues with Google maps.

Counting Murakami - 2015

counting-murakami.herokuapp.com

This is a website I created using Node.js and Express as a final project for a Japanese literature class on Haruki Murakami. I used Ruby regular expressions to gather word frequency data from text files of various Murakami novels and then created multiple visualizations based on that data using JavaScript graphing libraries.

The Universal Machine - 2014

This is an emulator for a simple computer that I programmed at the machine architect level with a partner for the class “Machine Structure and Assembly Language Programming.” The emulator is built using C and emulates a computer called the “Universal Machine” that has fourteen instructions and is Turing-complete.
