Abstract

This paper will examine the topic of Julia sets. In order to examine different Julia sets, the paper works through the definition of Julia sets and the basic equation of a Julia set. Next, the code written to generate Julia sets in matlab is explained in detail. This detailed examination of the code helps give a better understanding for Julia sets and how the graphs are made. Animated images of Julia sets are contained in this paper in order to explore how the graphs change based on a changing number of iterations. This paper examines different properties of the Julia sets such as the symmetry of the set and the connectedness of the set. At the very end of the paper, Basins of Attraction are explored along with their relationship to Julia Sets.

Now that the basis of Julia sets has been understood, this paper will explore the formation of Julia sets and how they change with changing c values.

Creating the image of a julia set for a given c value can be seen as taking the image of the Julia set at z2+0 (the unit circle), shifting it over by -c, and performing the same transformation on it many times. The transformation is made using the equation  and first halves the angle of the circle and then square-roots its magnitude. When the angle of a circle is halved, it is folded up halfway into a 180 degree fan. There are two of these such fans, one for the positive result of the square root and one for the negative result, that together make up one folded 360 degree fan. Finally, when the magnitude is square-rooted, the fan compresses towards a radius of 1 and expansion at the origin occurs. This describes the image after one round of fixed point iteration. As this transformation continues to be applied, a fractal image visualizing the Julia set begins to develop. In Matlab, this development of the Julia set as fixed point iteration takes place can be animated using a command called ‘drawnow’ and turned into a .gif file:

In the above graph, one can see that the Julia set goes from being an image where the white portion of graph is always connected to all the rest of the white portions to an image where portions of white are surrounded by areas of red. This is because Julia sets are either connected or disconnected. If they are disconnected then they are cantor sets meaning that it consists of infinitely many pieces. This was proved in 1919 by both Julia Gaston and Pierre Fatou (Devaney, 2006). In order to know if the Julia sets is connected or disconnected, one simply has to examine what happens for an initial value equal to zero. If when the initial value is zero and the fixed point iteration remains bounded, then Julia set is connected. If however the initial value is zero and the fixed point iteration approaches infinity, then the Julia set is disconnected (Lipa).

Another  interesting property of the Julia pertains to its symmetry. The symmetry of the set depends on the c value. If c is a real number then the Julia set has symmetry about the real axis. However, if the c value has a non-zero imaginary part then the Julia set has 180 degree rotational symmetry (Bourke, 2001). It can be easily seen above that in Figure 6 when all the c values have imaginary components the graph has rotational symmetry and that in Figure 7 when the c values do not contain an imaginary component the graph has symmetry about the real axis.

Above we noted how Julia Sets can either be connected or disconnected. Examining connected Julia Sets proves to be interesting because they can be thought of as Basins of Attraction, which are the sets of points in a region such that all initial values in that region will eventually be iterated to a certain attractor. Connected Julia sets can be thought of as Basins of Attraction because all initial values inside the white portion of the graph of a Julia set stay finite since fixed point iteration converges for those initial values. Thus we can say that the points inside the white region of the graph are attracted into a stable, finite orbit. All other initial values are repelled and tend to infinity as the fixed point iteration diverges (Fractal Foundation).         These graphs were created using MATLAB code from the MathWorks File Exchange written by Dr. Siva Srinivas Kolukula. The code works in a similar way to the code used in this paper to generate graphs of Julia Sets. It creates a matrix of initial values and performs Newton’s method on all points in the matrix. If Newton’s method of an initial value reaches one of the given roots, the point corresponding to that initial value is given a color corresponding to that root. If Newton’s method of an initial value does not reach one of the given roots, it is simply colored black.

clear; clf;

function f = jiteration (c, filename)
    % Initialize c, z, r, and the function

   A= 1.5;
   N=1000;
   x_vec=- A+[0:N]/ N*2*A; y_vec=x_vec;

    for i=1: length(x_vec)
       for  j=1:length(y_vec)
           x( i,j)=x_vec(i ); y(i,j)=y_vec( j);
       end;
    end;

   z = x+sqrt (-1)* y;
   f = @(z) z.^2  + c;

   % Iterate 25 function calls
   for kk=1 :25

       a= abs(z);
       a=min(a,1 );
       a=1- a;
       a=(a> 0);
       a=a +1;
       b = rot90(a);
       subplot( 111);
       image([ -A,A],[- A,A],b);
       colormap([ 1 0  0 ; 1  1  1 ]);
       axis([ -A,A, -A,A]);
       axis( 'square' );
       set(gca, 'Fontsize' , 24 );
       xlabel( 'x' , 'Fontsize' , 20 );
       ylabel( 'y' , 'Fontsize' , 20 );

       c_str=num2str(c);
        if  c > = 0
           title({[ 'Julia set for f(z) = z^2+ ' ,c_str];[ 'n = ' , num2str(kk)]});
        else
           title({[ 'Julia set for f(z) = z^2 ' ,c_str];[ 'n = ' , num2str(kk)]});
        end ;

       drawnow

        % Create .gif file
       F = getframe(gcf);
       im = frame2im(F);
       [Q,map] = rgb2ind(im, 256 );
        if  kk ==   1
           imwrite(Q,map,filename, 'gif' , 'LoopCount' ,Inf, 'DelayTime' , 0.35 );
        else
           imwrite(Q,map,filename, 'gif' , 'WriteMode' , 'append' , 'DelayTime' , 0.35 );
        end

       z = f(z);
    end
end
